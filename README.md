# Currency Rates Flow

A simple currency converter web application built using HTML, CSS, and JavaScript.

The application allows users to convert an amount from one currency to another using live exchange rate data and provides an easy-to-use interface for quick conversions.


## Features

- Convert between multiple international currencies
- Live exchange rate conversion
- Country flags for selected currencies
- Currency swap functionality
- Clean and responsive user interface
- Input validation for amount conversion


## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Exchange Rate API

1. Clone the repository

```bash

```

2. Open the project folder

3. Run `index.html` in your browser

No additional installation is required.
